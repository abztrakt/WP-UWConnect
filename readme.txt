[taxermlist] shortcode

== Description ==

= Shortcodes =

**[taxermlist tax="<taxonomy type>"]** - Lists out all the differnt taxonomy terms for the taxonomy type that you specify.
    When this short code is used, it will display a list of the terms with in the given taxonomy parameter. Each term in the list will be a link to a list of all the services associated with that taxonomy term.
