[taxermlist] shortcode

== Description ==

= Shortcodes =

**[taxermlist tax="<taxonomy type>"]** - Lists out all the differnt taxonomy terms for the taxonomy type that you specify.
    When this short code is used, the page will display a list of the taxonomies(categories) that the services are given. Each category in the list will be a link to a view of all the services that within that category.
